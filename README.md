# nsi-test

Каркас универсального модуля НСИ.

## Главное

Разработка ведется по `docs/NSI_MODULE_TARGET_SPEC.md`.

Этап 1 реализует:

- левое меню НСИ;
- центральное дерево выбранного раздела;
- правую карточку выбранного элемента;
- вкладки карточки;
- рабочую вкладку параметров;
- поиск, сортировку, раскрытие дерева и свернутость дерева по умолчанию;
- drag and drop для дерева объектов;
- рабочее меню действий строки дерева: добавить, редактировать, перенести, снять с учета, копировать;
- список добавления: дочерний объект, помещение, система, оборудование;
- UI-предупреждения вместо `window.alert` и `window.confirm`;
- чекбоксы связей, поиск по связям и массовый выбор найденных;
- универсальную модель object / объект инфраструктуры как базовую сущность НСИ.

Этап 2 добавляет:

- `objectTypes` в состоянии приложения, а не как неизменяемую глобальную константу;
- редактирование Дерева видов объектов в правой карточке;
- редактирование основных полей вида: наименование, код, сокращение, иконка, родитель, права создания/редактирования/снятия;
- чекбоксы допустимых дочерних видов с поиском и массовым выбором;
- редактор групп параметров;
- добавление, редактирование и удаление параметров вида;
- действия дерева видов: добавить дочерний вид, редактировать, перенести, снять с учета, копировать;
- выбор вида объекта при создании объекта учета.

Этап 3 добавляет:

- создание корневого объекта, дочернего объекта и помещения;
- динамические параметры объекта из `ObjectType.parameters`;
- поддержку `required` и `showInTree` параметров;
- перенос, копирование и снятие объектов с учета без физического удаления.

Этап 4 добавляет:

- шаблоны структуры объектов;
- создание корневого объекта пустым или из шаблона;
- универсальные demo-шаблоны: бизнес-центр, жилой комплекс, отель, склад, производственный объект;
- управляемый уровень детализации корневого объекта;
- отображение статуса детализации в карточке объекта и строке дерева.

## Структура интерфейса

```text
src/App.tsx                         контейнер состояния и бизнес-действий
src/components/layout/NsiLayout.tsx  общий трехпанельный layout
src/components/tree/NsiTree.tsx      центральное дерево и меню действий
src/components/details/DetailsPanel.tsx
src/components/details/ParameterContent.tsx
src/components/relations/RelationBlock.tsx
src/utils/nsiTree.ts                 helpers дерева, поиска, сводок и проверок
src/utils/nsiObjectParameters.ts     helpers динамических параметров объекта
src/utils/nsiObjectTemplates.ts      helpers шаблонов и уровня детализации
```
